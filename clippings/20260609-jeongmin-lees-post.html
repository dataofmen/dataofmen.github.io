에이전틱 엔지니어링, 매일 쓰는 22가지 전부 공개합니다

IDE 안 엽니다. 코드 직접 안 칩니다.

말하고 계획하고 빌드하는 루프만 돌리고 있습니다.

22가지 전부 정리했습니다.

코딩 전에 plan 부터 돌립니다. 스크린샷이든 에러 로그든 넣으면 리서치 에이전트가 병렬로 계획을 짜줍니다.

에이전트가 따라갈 체크 리스트지 사람이 읽을 문서가 아닙니다. 요약만 확인하고 빌드 돌리면 끝.

  1. 코드가 아닌 일에도 plan 을 씁니다

전략 문서나 제안서에서 “계획의 계획”을 먼저 세우면 에이전트가 깊은 결과물을 냅니다.

  1. 음성이 기본 입력 방식입니다

Wispr Flow 로 Codex 에 직접 말합니다. STT 정확도가 낮아도 LLM 이 문맥을 채워줘요.

  1. cmux 탭 4~6개를 동시에 운영합니다

한 탭에서 plan 도는 동안 다른 탭에서 build 하고 또 다른 탭에서 버그 잡습니다.

  1. 터미널을 열면 Codex 가 바로 뜹니다

새 탭이 shell 이 아니라 에이전트로 열리게 세팅하면 세션 시작 비용이 0이에요.

  1. 모든 세션을 원격에서 제어합니다

remote control 켜두면 모바일에서도 데스크 세션을 조작할 수 있고 AgentMail 로 이메일만 보내면 새 세션이 열립니다.

  1. dangerously skip permissions 는 필수입니다

6개 세션을 동시에 돌리면 일일이 승인할 수 없어요. 완료 알림 사운드만 걸어두면 됩니다.

  1. GPT Pro 가 계획하고 Codex 가 빌드합니다

Computer use 기반 ChatGPT 에서 GPT Pro 로 계획을 잡고, 해당 계획을 별도 파일을 저장해 Codex 로 빌드를 넘기면 두개의 서비스를 쉽게 병렬로 돌릴 수 있습니다.

  1. 계획 전에 리서치부터 합니다

last30days 로 Reddit, X, HN, YouTube 를 한번에 훑고 plan 을 만들면 최신 정보 기반으로 계획이 잡혀요.

  1. 미팅 녹취록은 허가를 받고 그대로 넣습니다

참여자 동의를 받은 뒤 Granola 녹취를 요약 없이 통째로 넣어요. 잡담은 알아서 걸러내고 핵심만 뽑아줍니다.

  1. 사람의 역할은 방향을 잡아주는 것입니다

에이전트가 볼륨을 만들고 사람이 방향과 감각을 줍니다. 손 대는 시간이 줄수록 더 많이 배포해요.

  1. 영상도 에이전트가 만듭니다

HyperFrames 로 영상을 HTML 처럼 작성하면 MP4 로 렌더링됩니다. 타임라인 편집기가 필요 없어요.

  1. 내 노트가 에이전트의 지식 저장소입니다

Notion 이나 Obsidian 에 쌓인 노트를 에이전트가 읽으면 세션마다 맥락이 누적돼요.

  1. VPS 하나면 어디서든 작업합니다

mosh + tmux 로 VPS 에 접속해두면 비행기에서 와이파이가 끊겨도 세션이 그대로 살아 있습니다.

GitHub 이나 공용 구글 드라이브에 올리면 비개발자도 깔끔하게 읽고 코멘트를 달 수 있어요.

  1. 반복하는 작업은 skill 로 만듭니다

기존 skill 구조를 복사하는 게 가장 빠릅니다. 두 번 이상 하는 작업은 전부 skill 로 만들어둬요.

  1. 오픈 소스에도 같은 방식으로 기여합니다

같은 plan + work 루프로 매일 쓰는 도구에 PR 을 보냅니다. merge 되면 maintainer 와 직접 협업이 생기고 실제 채용이나 프로젝트로 이어져요.

  1. 전원 확보는 기본 중의 기본입니다

6개 세션 + Codex 를 돌리면 M5 Pro 도 1시간 만에 배터리가 빠집니다. 휴대용 배터리 없이 외출 안 해요.

  1. 한국 생활도 K-Skill 로 자동화합니다

K-Skill 로 한국 생활에 맞는 자동화를 에이전트가 직접 실행합니다. Agent Cookie 가 브라우저 세션을 넘겨주는 게 핵심이에요.

  1. AI psychosis 는 진짜입니다

에이전트로 빌드하는 건 최고의 게임이지만 빠져들면 주변 사람을 잃어요. 쉬고 대화하세요.

  1. 이 글도 같은 방식으로 썼습니다

별도의 개발 환경, 키보드 입력 없이 말하고 계획하고 빌드하는 그 루프 만으로요.

꼭 필요한 AI 정보를 지속적으로 업로드하고 있습니다.

저와 “1촌”이 되면 유용한 정보를 놓치지 않을 거에요.
